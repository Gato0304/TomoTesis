Se utiliza una función escalón ya que está representa un cambio instantaneo la entrada de referencia, este hace que revele qué tan rapido responde un sistema a entradas con cambios abruptos.

Se debe tomar en cuenta algunos criterios de diseños utilizados para la caracterización de los sistemas de control lineales en el dominio del tiempo:


\begin{itemize}
	\item \textbf{Sobrepico máximo:} Si se define y(t) la respuesta del escalón unitario, $y_{ss}$ el valor en estado estable de y(t) y $y_{max}$ el valor máximo de la misma, entonces el sobrepico máximo se define como:
	
		\begin{center}
				$Sobrepico= y_{max}-y_{ss}$
		\end{center}
	
	El sobrepico se representa como un porcentaje del valor final, expuesto en la ecuación \ref{ec:sobrepico}:
	
		\begin{center}
		\begin{equation}\label{ec:sobrepico}
			\%Sobrepico=\dfrac{Sobrepico}{y_{ss}}*100
		\end{equation}
	\end{center}
	
	El sobrepico se utiliza para medir la estabilidad relativa del sistema de control por este motivo un  sobrepico alto es indeseable.
	
	\item \textbf{Tiempo de retardo $t_{d}$:} Es el tiempo que se necesita para que la respuesta escalón alcance el 50$\%$  del valor final de la gráfica.
	
   \item \textbf{Tiempo de alza $t_{r}$:} Es el tiempoo que toma la respuesta escalón se eleve del 10 al 90$\%$ del valor final.
   
   \item \textbf{Tiempo de establecimiento $t_{s}$:} Es el tiempo que se requiere para que la respuesta escalón  disminuya y permanezca dentro de un rango del valor final. Un valor frecuente que se utiliza es 5$\%$ y se calcula \ref{ec:ts}.
   
   \begin{center}
   	\begin{equation}\label{ec:ts}
   		 T_{s}=3\tau=\frac{3}{\zeta\omega_{n}}
   	\end{equation}
   \end{center}
   
\end{itemize}

Como ya se ha mencionado en los capítulo \ref{CAP:tecni} los sistemas de control de segundo orden con realimentación unitaria a lazo cerrado se representa en la ecuación \ref{ec:segundoorden}


\begin{center}
	$	F(s)=\frac{\omega_{n}^{2}}{s^{2}+2\zeta \omega_{n}s+\omega_{n}^{2}}$
\end{center}

El factor de amortiguamiento $\zeta$ y la frecuencia natural $\omega_{n}$ constituyen las raíces de la ecuación característica. Cuando $\zeta \geq 1$ la respuesta escalón no muestra sobrepicos,  si $0< \zeta < 1$ entonces el sistema es subamortiguado y si $\zeta=o $ el amortiguamiento es cero, el sistema se vuelve críticamente estable. Las dos raíces se pueden expresar:

\begin{center}
	\begin{equation}\label{raices}
		S_{1},S_{2}=-\zeta\omega_{n}\pm j\omega_{n}\sqrt{1-\zeta^{2}}
	\end{equation}
\end{center}
