Como recomendación principal se propone mejorar este prototipo para crear prácticas de laboratorio más actualizadas, con el objetivo de que los estudiantes puedan tener acceso a distancia.

También se recomienda seguir esta metodología para recrear las prácticas y objetivos basados en controladores PID (Proporcional, Integral y Derivativo). 

Por otra parte, se pueden implementar y agregar otros tipos de controladores para el sistema y a la interfaz gráfica, obteniendo de esta manera más opciones de respuesta para controlar la salida del sistema.

Para probar una mejora en la aproximación decimal del resultado final se puede usar otro tipo de soluciones numéricas, por ejemplo el método de Newton-Raphson, y un mayor número de decimales para las operaciones.

Otra forma de realizar este trabajo es diseñando un módulo de adquisición de datos analógicos que permitan modelar la señal del sistema y el controlador con una mejor precisión.
